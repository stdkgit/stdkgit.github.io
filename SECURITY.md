# Security Policy

## Supported Versions

This project does not have specific versioned releases. All updates are applied directly to the main branch.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please contact us at **iintegre@proton.me**. We aim to respond as quickly as possible.  
Do **not** open a GitHub issue for security problems; use the email above.

When you report a vulnerability, please include:  
- A clear description of the issue  
- Steps to reproduce the problem  
- Any relevant screenshots, logs, or code snippets  

You will receive updates on the status of your report, and we will let you know if the vulnerability is accepted or declined.
